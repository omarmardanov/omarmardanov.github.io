## 1. Отраслевая разбивка патентов в меню «Услуги»

### Обсуждение
Заказчик отметил: одна из сильных сторон фирмы — **патентная практика по отраслям**:
- Сельское хозяйство (с/х изобретения)
- Медицина и медтехника
- Фармацевтика (включая фармакологические соединения)
- Нефтегаз
- Промышленное оборудование

На старом сайте 1-tm.ru есть отдельные узкоспециализированные страницы услуг типа «Патент изобретения для нефтегазовой промышленности» — они **созданы под low-frequency SEO-запросы** и хорошо ранжируются по узким темам.

### Вопрос
Как вывести эту специализацию в меню «Услуги», не превращая мегаменю в простыню?

### Предложенные направления для будущего обсуждения

**Вариант A: Доп. колонка в мегаменю «По отраслям»**
В мегаменю 4 колонки — направления. Добавить **5-ю колонку «Патенты по отраслям»** (или встроить как подменю в колонку ИС → Патенты).

```
[ ИС ]  [ Договоры ]  [ Суд. практика ]  [ Сопровождение ]  [ Патенты по отраслям ]
                                                              · С/х
                                                              · Медицина
                                                              · Фарма
                                                              · Нефтегаз
                                                              · Промышленность
```

Плюсы: видимая презентация специализации. Минусы: усложняет UX, нарушает симметрию 4 направлений.

**Вариант B: Промо-блок внизу мегаменю**
Под четырьмя колонками направлений — горизонтальная полоса с тегами отраслей: «Патенты в фарме · Патенты в нефтегазе · …» — каждый тег ведёт на отфильтрованную страницу.

**Вариант C: Отдельная страница `/uslugi/patenty-po-otraslyam/`**
Не в меню, но с прямой ссылкой из:
- блока «Патенты» внутри направления ИС
- главной (eyebrow в hero «Также сильны в патентах для фармы, нефтегаза…»)
- статей-обзоров

**Вариант D: Тег `industries` на услугах**
В WP завести таксономию `industries`. На страницах услуг показывать чипы «Применимо в: фарма, нефтегаз». Каждый тег → отдельная страница `/otrasli/farma/`.

### Текущая позиция
Объяснили заказчику: low-frequency запросы хорошо обрабатываются через **поиск сайта и поисковые системы (Google/Яндекс)** — пользователь либо вводит «патент в нефтегазе» в Яндекс и попадает на узкую страницу, либо в поиске сайта. Демонстрировать эти 20+ узких услуг **в меню — переусложнение**.

Но идею не закрываем — заказчик хочет вернуться. Предложение для итерации: **Вариант D + Вариант C** (теги + отдельная посадочная). Это даёт SEO-индексацию, не перегружает меню, и при этом отдельный landing для прямой рекламы.

## 2. Подсветка судебной практики по специализациям

### Обсуждение
Заказчик: компания сильна в судах разных инстанций и специализаций — Арбитражный суд, СИП, Палата по патентным спорам, ФАС, Верховный Суд РФ. Хочется чтобы это было видно.

### Вопрос
Как подчеркнуть глубину судебной практики, не превращая страницу направления в скучный список инстанций?

### Идеи на потом

**Idea 1: «Судебный счётчик» в hero направления «Судебная практика»**
Под заголовком — табло с цифрами:
```
СИП — 47 представительств · Арбитраж — 312 дел · ФАС — 28 решений · Палата ППС — 84 заседания
```

**Idea 2: Группировка кейсов на /dela/ по инстанциям**
Дополнительно к фильтру по типу спора (ТЗ, патенты, налоги) — фильтр по инстанции. Заходишь на /dela/?suit=arbitrazh — видишь только арбитражные кейсы.

**Idea 3: Раздел «Где мы выступаем» внутри /uslugi/spory-i-zashchita/**
Отдельная секция с 5–6 блоками: СИП, Арбитраж, ФАС, и т.д. — у каждого: что разбирают, наш опыт, типовые кейсы.

**Idea 4: Отраслевые подразделы судебной практики**
По аналогии с патентами по отраслям — Налоговая практика в фарме, Корпоративные споры в нефтегазе и т.п. Это уже выходит за рамки текущей структуры, но даёт сильное SEO.

## 3. Прочие зацепки

- **Карты офисов** — сейчас плейсхолдеры с диагональной штриховкой. Подключить Яндекс.Карты или статичные изображения карт.
- **EN-версия сайта** — отложена, но в разметке уже учитываем (lang="ru" + canonical/alternate можно будет легко добавить).
- **«СМИ о нас»** — убрали с главной по запросу, но если потребуется потом — можно вернуть отдельной страницей `/smi/`.
- **Партнёр в каждом деле / Прозрачные гонорары / Никаких шаблонов** — три принципа сейчас в /o-byuro/. Можно вынести как отдельную микро-секцию на главной, перед CTA.
