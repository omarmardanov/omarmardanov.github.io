# Content templates — гайд для заказчика

Папка с шаблонами для подготовки контента сайта 1-tm.ru.

**Адресат:** контент-команда заказчика (юристы, маркетинг, ассистенты партнёров).
**Цель:** довести материалы до формы, в которой WP-разработчик может загрузить
их в админку без переписывания и угадывания.

---

## С чего начать

1. **Прочитать [00-pravila-i-stil.md](00-pravila-i-stil.md)** — единый гайд по
   тону, цифрам, italic-акцентам, типографике, чек-листам. Без этого шаблоны
   ниже будут заполняться по-разному, и WP-разработчик будет переписывать.

2. **Выбрать шаблон под тип материала** (см. таблицу ниже).

3. **Заполнять прямо в копии файла** — например, `01-usluga.md` копируете
   в `content/uslugi/registratsiya-tovarnogo-znaka.md` и заполняете
   плейсхолдеры на свой текст.

## Очерёдность подготовки (рекомендация)

Если ресурс ограничен — приоритеты:

1. **3 направления** (`dogovory-i-sdelki`, `spory-i-zashchita`, `soprovozhdenie-biznesa`) — это основной путь пользователя. Без них каталог пустой. → [02-napravlenie.md](02-napravlenie.md)
2. **10 специализаций** (отраслевые + судебные лендинги) — пункты меню уже ведут на эти URL, и они сейчас отдают 404. Это видимая проблема. → [03-spetsializatsiya.md](03-spetsializatsiya.md)
3. **Топ-20 услуг по запросам** — те, по которым больше всего обращений. → [01-usluga.md](01-usluga.md)
4. **Топ-10 кейсов** — наиболее показательные, с согласием клиента. Без кейсов карточки на главной выглядят пусто. → [04-keys.md](04-keys.md)
5. **Все 45 классов МКТУ** — таблицей, наполняется быстро. → [06-mktu-klass.md](06-mktu-klass.md)
6. **Топ-5 статей журнала** — по самым горячим темам. → [05-statya.md](05-statya.md)
7. **О бюро + команда** — карточки сотрудников. → [07-sotrudnik.md](07-sotrudnik.md)

Контент можно готовить параллельно разными людьми — шаблоны самостоятельные.

## Куда сдавать готовое

Папка на общем диске:
```
content/
  uslugi/                    ← по одному .md/.docx на услугу
  napravleniya/              ← 3 файла
  spetsializatsiya/          ← 10 файлов
  dela/                      ← кейсы, по одному файлу
  zhurnal/                   ← статьи, по одному файлу
  mktu/                      ← одна таблица Excel на все 45 классов
  komanda/                   ← один файл на сотрудника или общая таблица
  photos/                    ← все картинки, разложены по подпапкам
    team/
    zhurnal/
    dela/
    clients/                 ← SVG-логотипы клиентов
  consents/                  ← согласия клиентов на публикацию кейсов (PDF)
```

**Имя файла = slug** (например, `registratsiya-tovarnogo-znaka.docx`).

Финальный реципиент — WP-разработчик. Адрес и контакт — у заказчика.

## Если возник вопрос

1. Сначала проверьте [00-pravila-i-stil.md](00-pravila-i-stil.md) — там скорее
   всего есть ответ (особенно про цифры, кавычки, тире, штампы).
2. Затем — чек-лист в конце соответствующего шаблона.
3. Если не нашли — пишите редактору сайта или WP-разработчику.

Шаблоны самостоятельны, но **гайд правил выше всего**. Если шаблон что-то
не оговорил, а в `00-pravila-i-stil.md` оговорено — действует правило из гайда.
