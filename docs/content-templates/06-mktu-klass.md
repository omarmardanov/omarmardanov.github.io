> Всего 45 классов — 1 страница на класс.
>
> **Эталон на сайте:** [/mktu/klass-41/](https://1-tm.ru/mktu/klass-41/)
> **URL результата:** `/mktu/klass-{N}/` (где N от 1 до 45)
>
> Прочитайте сначала [00-pravila-i-stil.md](00-pravila-i-stil.md).

---

## Особенности этого типа

В отличие от услуг и статей, **контент класса МКТУ — официальный**. Его не нужно
сочинять, а нужно **корректно перенести** из источника (Роспатент, Ниццкое
соглашение в действующей редакции). Творческая часть — только связанные услуги
и кейсы.

**Источник истины:** [https://www.fips.ru/](https://www.fips.ru/) и официальная
действующая редакция МКТУ (на 2026 год — 12-я редакция Ниццкой классификации).

**Удобный способ:** заполнить **одной таблицей Excel/Google Sheets** все 45 классов
сразу — поля 1–4. Затем редактор сайта добавляет поля 5–6 для каждого класса.

## 1. SEO-поля

### 1.1 `<title>` (до 65 символов)

Формула: `Класс N МКТУ — [краткое название] · Галифанов, Мальков`

**Примеры:**
- `Класс 35 МКТУ — реклама, менеджмент · Галифанов, Мальков`
- `Класс 41 МКТУ — образование, развлечения · Галифанов, Мальков`

### 1.2 Meta description (130–160 символов)

```
Класс N Международной классификации товаров и услуг (МКТУ) — [краткое описание сферы]. Полный перечень + услуги по регистрации товарного знака в этом классе.
```

**Пример (класс 41):**
`Класс 41 МКТУ — образовательные услуги, проведение тренингов и семинаров, организация развлечений, спортивных и культурных мероприятий. Полный перечень + регистрация ТЗ в этом классе.`

## 2. Поля для каждого класса

### 2.1 Номер класса

```
N (от 1 до 45)
```

### 2.2 Название класса (официальная формулировка)

Берётся из официальной редакции МКТУ. Не переписывать своими словами.

**Пример (класс 41):**
`Воспитание; обеспечение учебного процесса; развлечения; организация спортивных и культурно-просветительных мероприятий.`

### 2.3 Краткое описание (50–150 символов, своя формулировка)

Это **наша** формулировка — для отображения в каталоге `/mktu/` и в карточке класса. Должна быть понятной не-юристу.

**Пример (класс 41):**
`Образование, тренинги, развлечения, спортивные и культурные мероприятия.`

**Пример (класс 35):**
`Реклама, маркетинг, бизнес-консалтинг, менеджмент, услуги розничной торговли.`

### 2.4 Полный перечень товаров/услуг (официальный)

Полный список из официальной редакции МКТУ. Источник — Роспатент. Не сокращать, не переписывать. Часто длинный (50–200 пунктов на класс).

**Формат:** маркированный список или нумерованный.

**Пример (фрагмент класса 41):**
- образование
- проведение тренингов
- проведение семинаров
- организация и проведение конференций
- организация и проведение симпозиумов
- организация и проведение конкурсов учебных или развлекательных
- ...

Полный список можно скопировать с rospatent.gov.ru.

### 2.5 H1 страницы

Стандартный формат, не творить:

```
Класс N МКТУ
```

**Italic-акцент** — на номере: `Класс *N* МКТУ`. Реализуется автоматически шаблоном.

### 2.6 Подзаголовок (под H1, опционально)

Краткое описание сферы класса, чтобы посетитель сразу понял, его ли это класс.

**Пример (класс 41):**
`Все услуги, связанные с образованием, развлечениями, спортом и культурой.`

## 3. Связанные услуги (заполнить или пометить «редактор»)

3–5 услуг бюро, релевантных этому классу.

```
Связанные услуги:
- [Название услуги] (/uslugi/...)
- [Название услуги] (/uslugi/...)
```

**Пример (класс 41):**
- Регистрация товарного знака в Роспатенте (/uslugi/intellektualnaya-sobstvennost/registratsiya-tovarnogo-znaka/)
- Поиск по базе Роспатента (/uslugi/intellektualnaya-sobstvennost/poisk-po-tz/)
- Возражение против отказа в регистрации (/uslugi/intellektualnaya-sobstvennost/vozrazhenie-na-otkaz/)

Если не уверены — пометьте «подбирает редактор».

1–3 кейса, где регистрировали или защищали товарный знак в этом классе.

```
Релевантные кейсы:
- [Название кейса] (/dela/...)
```

## 5. CTA-баннер на странице класса

Не CTA-final (на этих страницах он заменён на узкий баннер `.mktu-cta`). Текст под класс.

**Пример (класс 41):**
```
Заголовок: Нужно зарегистрировать ТЗ в классе 41?
Текст: Поможем подобрать формулировки, проверить наличие препятствий и подать
заявку под образовательные или развлекательные услуги.
Кнопка: Заявка на консультацию
```

Если хочется единый текст на все 45 классов — допустимо. Тогда WP-разработчик использует шаблон с подстановкой номера.

**Универсальный текст:**
```
Заголовок: Регистрация товарного знака в классе [N]
Текст: Подадим заявку с правильным перечнем по классу [N], сопроводим экспертизу
Роспатента и получим свидетельство. Срок 6–10 месяцев.
Кнопка: Заявка на консультацию
```
