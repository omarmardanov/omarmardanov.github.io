# Контент-гайд 1-tm.ru

Документация для контент-команды бюро: как готовить статьи, кейсы, услуги и другие материалы для сайта так, чтобы они уходили в публикацию без переписывания.

**Кто читает:** маркетолог, контент-менеджер, юристы, ассистенты партнёров, копирайтеры — все, кто пишет тексты для сайта.
**Кто получает готовое:** WP-разработчик, загружает в админку.

---

## С чего начать

1. **[Старт за 10 минут](Старт за 10 минут.md)** — если открываете гайд впервые.
2. **[Правила и стиль](Правила и стиль.md)** — тон голоса, цифры, штампы, типографика. **Читать всем перед первым заполнением шаблона.**
3. **[Шаблоны](Шаблоны.md)** — структура полей под каждый тип страницы (услуга, направление, специализация, кейс, статья, МКТУ-класс, сотрудник).
4. **[Примеры заполненных файлов](Примеры.md)** — как выглядит готовый материал, который вы сдаёте. Сверяйтесь со своим черновиком.
5. **[Чек-лист перед публикацией](Чек-лист перед публикацией.md)** — финальная проверка перед сдачей.
6. **[Процесс и контакты](Процесс и контакты.md)** — куда сдавать готовое, кто согласует.

## Что важно знать сразу

- **Тон голоса:** сухо, конкретно, по делу. *Юрист, разговаривающий с клиентом-собственником бизнеса.* Не маркетолог, не лектор.
- **Цифра побеждает прилагательное.** «12,4 млн ₽ компенсации» вместо «крупная компенсация».
- **«Мы» / «бюро», обращение на «вы».** Не «компания», не «фирма», не «наша команда».
- **Чёрный список штампов** в «[Правилах и стиле](Правила и стиль.md)» — слова оттуда никогда не пишем.
- **Формат сдачи:** один файл = одна страница сайта. Имя файла = URL-slug (например, `registratsiya-tovarnogo-znaka.docx`).

## Приоритеты подготовки

Если ресурс ограничен — порядок:

1. **3 направления** (договоры и сделки, споры и защита, сопровождение бизнеса) — основной путь пользователя.
2. **10 специализаций** (отраслевые + судебные лендинги) — пункты меню уже ведут на эти URL, и они отдают 404.
3. **Топ-20 услуг по запросам** — те, по которым больше всего обращений.
4. **Топ-10 кейсов** — наиболее показательные, с согласием клиента.
5. **Все 45 классов МКТУ** — таблицей, наполняется быстро.
6. **Топ-5 статей журнала** — по горячим темам.
7. **Команда** — карточки сотрудников.

Готовить можно параллельно разными людьми — шаблоны самостоятельные.
