## Как пользоваться шаблоном

1. Открыть нужный шаблон (ссылки в таблице выше).
2. **Открыть эталон на сайте в соседней вкладке** — видеть длину, тон, плотность.
3. Скопировать структуру в свой документ (`Duplicate` в Notion, или вставить в Word/Google Docs).
4. Заполнять по порядку сверху вниз — не пропускать секции.
5. В конце шаблона пройти чек-лист.
6. Сдать в работу по процессу из «[Процесс и контакты](Процесс и контакты.md)».

## Общие правила, действующие для всех шаблонов

- **Имя файла = URL-slug.** Только латиница, дефисы вместо пробелов, без заглавных. Пример: `registratsiya-tovarnogo-znaka`.
- **Один материал = один файл.** Не складывайте несколько услуг в один документ.
- **Картинки — отдельной папкой**, имена = slug + порядковый номер.
- **Если поля нет в шаблоне** — значит, его нет на сайте. Не добавляйте «от себя» — это сломает верстку.
- **Если поле есть, но вы не знаете чем заполнить** — пишите редактору, не пропускайте и не угадывайте.
