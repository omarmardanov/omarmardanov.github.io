# Чек-лист перед публикацией

Финальная проверка перед сдачей материала. Если хотя бы один пункт «нет» — возвращаем на доработку, не публикуем.

---

## Контент

- [ ] **Тон голоса** — сухо, по делу, цифры вместо прилагательных. Нет ни одного слова из чёрного списка ([Правила и стиль § 2](Правила и стиль.md))
- [ ] **Лид (первый абзац)** отвечает на «зачем читать дальше» за 2–3 строки
- [ ] **Цифры расставлены** — сроки, цены, объёмы, проценты — конкретно, а не «быстро/много/выгодно»
- [ ] **Нет канцелярита и пресс-релизного тона** — фразы можно произнести вслух не закатывая глаза
- [ ] **Названия — точные**, без вариаций (одно и то же бюро = «бюро» во всём тексте, не «компания», не «фирма»)
- [ ] **Источники указаны** — где цитата из закона, ссылка на статью; где статистика — источник
- [ ] **Длина соответствует эталону** — открыли эталон-страницу, сравнили объём, не сократили вдвое и не растянули в три раза

---

## Структура

- [ ] **Все обязательные секции шаблона заполнены** — ничего не пропущено, даже если кажется второстепенным
- [ ] **Заголовки иерархически правильные** — нет «прыжков» (H1 → H3 без H2)
- [ ] **Список «Что входит / Что не входит»** заполнен парами, не перекошен
- [ ] **CTA-блок** — есть призыв, текст кнопки точный (не «узнать больше», а «обсудить регистрацию»)

---

## SEO

- [ ] **`<title>`** — до 65 символов, содержит главный запрос, не повторяет H1 буква-в-букву
- [ ] **`<meta description>`** — 140–160 символов, передаёт пользу, не штампы
- [ ] **H1** — один на странице, отличается от title
- [ ] **URL-slug** — латиница, дефисы, без заглавных, читается как тема
- [ ] **Alt у всех картинок** — описывают что на картинке, а не «изображение №1»

---

## Юридическое

- [ ] **Кейс — есть согласие клиента** на публикацию (PDF в папке `consents/`)
- [ ] **Если упоминаются клиенты/контрагенты по именам** — есть согласие
- [ ] **Цитат закона** — со ссылкой на статью и редакцию
- [ ] **Если упоминаются конкретные дела** — номер дела + ссылка на kad.arbitr.ru или соответствующий ресурс
- [ ] **Нет оферты** — нигде в тексте материал не выглядит как публичная оферта на услугу (цены — справочные, не обязательство)

---

## Картинки и медиа

- [ ] **Обложка статьи / кейса** — минимум 1600 px по широкой стороне
- [ ] **Без вотермарков и копирайтов** на самом изображении
- [ ] **Имена файлов = slug + порядковый номер** (`registratsiya-tovarnogo-znaka-01.jpg`)
- [ ] **Фото сотрудника** — 1000×1000 px, JPG, центрированное лицо

---

## Передача WP-разработчику

- [ ] **Файл назван slug**’ом, не «Новый документ (3)»
- [ ] **Все картинки приложены** в подпапке, имена соответствуют
- [ ] **В шапке файла** заполнены: тип, slug, направление, автор, дата, статус: «готово»
- [ ] **Чек-лист в конце шаблона** пройден полностью

---

После прохождения чек-листа — переводите статус материала в «Готов к публикации» и передавайте по процессу из «[Процесс и контакты](Процесс и контакты.md)».
