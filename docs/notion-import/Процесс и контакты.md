# Процесс и контакты

Как готовый материал доходит до публикации, кто за что отвечает.

---

## Цикл материала

```
1. Идея / запрос
   ↓
2. Бриф (выбран тип, эталон открыт)
   ↓
3. Черновик (заполнен шаблон)
   ↓
4. Самопроверка (пройден чек-лист в шаблоне + общий)
   ↓
5. Редакторская правка
   ↓
6. Юридическое согласование (если кейс / упоминание клиентов)
   ↓
7. Передача WP-разработчику
   ↓
8. Публикация на сайте
```

Статус материала в Notion отражает текущий шаг: `черновик` → `на правке` → `на согласовании` → `готов` → `опубликовано`.

---

## Кто что делает

| Роль | Зона ответственности |
|---|---|
| **Автор** (юрист / маркетолог / копирайтер) | Заполняет шаблон, проходит чек-лист, передаёт на редактуру |
| **Редактор сайта** | Правит тон, цифры, штампы; возвращает доработку или принимает |
| **Юрист-предметник** | Согласует фактологию по своей теме (ТЗ, патенты, суды) |
| **Партнёр** | Финальное согласование кейсов и публикаций с упоминанием клиентов |
| **WP-разработчик** | Получает готовое, размещает в админке. Текст не переписывает. |

---

## Куда сдавать готовое

### Файлы материалов
Папка на общем диске:
```
content/
  uslugi/                    ← по одному файлу на услугу
  napravleniya/              ← 3 файла
  spetsializatsiya/          ← 10 файлов
  dela/                      ← кейсы
  zhurnal/                   ← статьи
  mktu/                      ← одна таблица на все 45 классов
  komanda/                   ← по одному файлу на сотрудника
  photos/                    ← картинки по подпапкам
    team/
    zhurnal/
    dela/
    clients/                 ← SVG-логотипы клиентов
  consents/                  ← согласия клиентов на публикацию кейсов (PDF)
```

**Имя файла = slug** (например, `registratsiya-tovarnogo-znaka.docx`).

### Согласия клиентов (для кейсов)
PDF-скан подписанного согласия — в `content/consents/`. Имя файла = slug кейса.

## SLA (ориентир)

| Шаг | Целевой срок |
|---|---|
| Редакторская правка | 2 рабочих дня |
| Юридическое согласование (статья) | 3 рабочих дня |
| Юридическое согласование (кейс) | до 5 рабочих дней |
| Публикация WP-разработчиком | 1 рабочий день после получения готового |

Если срок проседает — пишите редактору в указанный канал.

---

## Если что-то пошло не так

- **Не уверены какой шаблон выбрать** → редактор.
- **Не можете найти нужное правило** → проверьте [Правила и стиль](Правила и стиль.md), затем — редактор.
- **Юрист не согласовывает фактологию** → редактор, не публикуем спорные утверждения.
- **Клиент не дал согласие на кейс** → кейс не публикуется, либо обезличивается до неузнаваемости (отрасль/масштаб/результат — без имени и узнаваемых деталей).
