## Эволюция концепции

**Что было:** на старом сайте 1-tm.ru фирма позиционировалась как **«Федеральное патентное бюро»**, фокус почти исключительно на интеллектуальной собственности.

**Что есть сейчас:** ассортимент расширен до полноценной юридической фирмы — добавлены налоги, корпоративные споры, банкротство, госзакупки, сопровождение бизнеса, антимонопольная защита, аутсорсинг.

**Аудитория:** бизнес (МСП, средний и крупный), директора, юр. отделы, изобретатели, стартапы, иностранные компании, выходящие на рынок РФ.

## Сильные стороны / экспертиза

### Патентная практика по отраслям
Бюро сильно в патентовании в **отраслевых нишах** — это даёт SEO-преимущество по low-frequency запросам:
- Сельское хозяйство (с/х изобретения)
- Медицина и медтехника
- Фармацевтика (фармакологические изобретения, химические соединения)
- Нефтегаз (технологии добычи, переработки)
- Промышленное оборудование

Подробнее о вариантах подсветки этой специализации в меню — см. `docs/FUTURE_IDEAS.md`.

### Судебная практика
Сильны в судах разных инстанций и специализаций:
- Арбитражные суды всех инстанций
- Суд по интеллектуальным правам (СИП)
- Палата по патентным спорам Роспатента
- ФАС России (антимонопольные дела)
- Верховный Суд РФ
- Налоговые суды

Идеи отраслевой/инстанционной разбивки — `docs/FUTURE_IDEAS.md`.

### Метрики (с 1-tm.ru)
- 1300+ зарегистрированных товарных знаков
- 500+ защищённых клиентов
- 90,4% выигранных дел
- #7 в рейтинге IP-фирм РФ

## Структура услуг — концепция «4 направления»

На сайте 1-tm.ru сейчас услуги — **плоский список из ~18 пунктов** без группировки. Мы предлагаем заказчику новую структуру: **4 направления → разделы → услуги**.

### Маппинг 18 услуг 1-tm.ru → 4 направления

| Направление | URL | Услуги |
|---|---|---|
| **Интеллектуальная собственность** | `/uslugi/intellektualnaya-sobstvennost/` | Товарные знаки, Патенты, Авторское право, ПО и БД, Географические указания, Наименования мест происхождения, Франшиза |
| **Договоры и сделки** | `/uslugi/dogovory-i-sdelki/` | Лицензионные договоры, Договоры отчуждения, Договоры залога, Коммерческая тайна / NDA, Авторский заказ, Концессия |
| **Судебная практика** | `/uslugi/spory-i-zashchita/` | Арбитражные споры, Корпоративные споры, Налоговые споры, ФАС / Антимонопольная защита, Борьба с контрафактом, Банкротство |
| **Сопровождение бизнеса** | `/uslugi/soprovozhdenie-biznesa/` | Юридическое сопровождение, Госзакупки 44/223-ФЗ, Бизнес-аутсорсинг (бухгалтерия, переводы), Защита бизнеса |

URL `/uslugi/spory-i-zashchita/` сохраняем (для SEO-стабильности и совместимости), название в UI — **«Судебная практика»**.

> **Реализация направлений** в WP — см. `docs/WORDPRESS.md`. На фронте плейсхолдеры пометки `<!-- WP: loop ... -->`.

## Технические требования
- Ванильный HTML + CSS + JS, без фреймворков и сборщика
- Каждая страница — самодостаточный HTML-файл, ссылающийся на shared CSS/JS из `/assets/`
- Адаптивность: desktop (1180px+), tablet (768–1024px), mobile (≤640px)
- Семантическая разметка, aria-атрибуты
- Смена языка (RU/EN) пока **не реализуется** — сайт только на русском
- Страницы услуг (~80 шт.) создаются в WordPress по шаблону `/uslugi-shablon/`
- Классы МКТУ (45 шт.) — в WP по шаблону `/mktu/klass-41/`
- В шапке нет ссылки на «Классы МКТУ» — справочник доступен через футер и поиск
