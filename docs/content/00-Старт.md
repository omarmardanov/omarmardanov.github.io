# Старт

Вы готовите тексты для сайта 1-tm.ru. Этот гайд — всё, что нужно: правила, шаблоны, куда сдавать.

**Сайт:** [1-tm.ru](https://1-tm.ru) — открывайте, чтобы видеть, как тексты выглядят в проде.
**Получатель готового:** WP-разработчик, загружает в админку без переписывания.

---

## Что делать

1. Прочитать **[01-Правила и стиль](01-Правила%20и%20стиль.md)** — один раз перед началом.
2. Выбрать шаблон под тип материала:
   - **Услуга** (110+ страниц, включая отраслевые/судебные лендинги) → [02-Шаблон услуги](02-Шаблон%20услуги.md)
   - **Направление** (3 страницы) → [03-Шаблон направления](03-Шаблон%20направления.md)
   - **Статья журнала** → [04-Шаблон статьи](04-Шаблон%20статьи.md)
   - **Кейс / дело** → [05-Шаблон кейса](05-Шаблон%20кейса.md)
   - **Сотрудник** → [06-Шаблон сотрудника](06-Шаблон%20сотрудника.md)
3. Скопировать шаблон в новый документ, заполнить поля.
4. Перед сдачей пройти **[08-Чек-лист](08-Чек-лист.md)**.

**Что не пишем заново**, а переносим с текущего сайта — см. **[07-Перенос с прода](07-Перенос%20с%20прода.md)**.

---

## Формат сдачи

- Один документ = одна страница сайта.
- Имя документа = URL-slug страницы (например, `registratsiya-tovarnogo-znaka`).
- Фото и SVG-логотипы — отдельной папкой, имя файла = slug страницы или сотрудника.
