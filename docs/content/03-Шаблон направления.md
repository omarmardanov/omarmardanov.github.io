**`<title>`** (≤ 65 символов): `{Название} — услуги бюро · Галифанов, Мальков и Партнёры`

## 3. Разделы и услуги

Услуги группируются в **разделы** — это визуальная группировка внутри страницы направления. **URL услуг остаются плоскими** (`/uslugi/{slug}/`), раздел не уходит в URL.

Формат — для каждого раздела:

```
Раздел: Корпоративные договоры
Описание раздела (1 предложение, 50–150 симв.): Подготовка и проверка договоров для повседневной работы бизнеса.

Услуги в разделе (4–10):
- Договор поставки — подготовка с защитой ваших интересов, риск-анализ типовых пунктов.
  Slug: dogovor-postavki
- Договор оказания услуг — проработка предмета, ответственности, порядка приёмки.
  Slug: dogovor-uslug
- ...
```

В одном направлении обычно 3–6 разделов.
